# Projeto EncontraSala<br>

Software baseado em web de página singular que auxilia no gerenciamento da alocação de salas de aula existentes em universidades. Objetivado para facilitar o agendamento de horários para utilização de salas, dispondo de um sistema de perfis, permitindo que cada usuário controle seus agendamentos.<br>

## Recursos Básicos
<ul>
<li>Visualização global e individual da disponibilidade de salas por horário;<br>
<li>Listagem de salas com informações;<br>
<li>Seleção e adição de usuário;<br>
<li>Medidas de segurança;<br>
<li>Cadastro e remoção de salas.
</ul>

## Tecnologias Utilizadas

O projeto foi construído usando HTML5, CSS3 e JavaScript.
